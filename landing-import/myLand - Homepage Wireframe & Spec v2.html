Interaction behaviors Performance-first. All motion is transform/opacity only, rAF-throttled, and disabled under prefers-reduced-motion (static fallback that still looks composed).

1 Topbar

Transparent over the hero; gains a glass/blur background + hairline border once the user scrolls past the fold. The identity dot pulses slowly (~3s), matching the login screen.

2 Hero canvas — parallax & depth

Hand-curated layout. Layers translate at different rates following the cursor (depth illusion); 1–2 elements use a slow float loop; the presence dot pulses. No real drag — a render, tuned for a strong, instant first impression.

3 Handle field

Type a handle → live availability check (reuses login validation). Mint state + ✓ when free, red hint when taken/invalid. Enter or button → register flow with the handle pre-filled.

4 Hero — list blooms into a space

On load (and again on re-entry), a rigid stack of grey link-rows scatters and reassembles into the curated canvas. This delivers the anti-list contrast inside the hero — no separate "vs." section needed. Reduced-motion: canvas simply appears, list ghost stays faint.

5 Kit pieces

Loose blocks float at slightly different phases. Hover lifts + straightens a piece (rotate→0, small scale), hinting these are movable objects, not menu items.

6 Guestbook wall

Entries appear to accumulate: on reveal they drop/fade in one by one, slightly rotated, like pinned scraps. Optional live tick adds a new mark every few seconds to sell "spaces gather history." Presence dot + visit counter animate up.

7 Gallery cards

Reuses the existing BrowseCard / CanvasPreview: live mini-canvas, cursor parallax, hover scale (~1.04) + brightness. Click → /[handle]. Seed profiles fill in until enough real, high-quality spaces exist.

8 Pricing

Calm 3-tier band (Plot · Acre · Estate-soon). Hover lifts a card; the recommended tier carries the mint accent. Structured so new tiers/feature rows drop in without a redesign.

9 Guestbook trace in the hero

The curated hero canvas already carries the guestbook thread: a couple of visitor scraps (“alex was here ♥”, a gif), a live 3 here now dot and a 12,408 visits count. Visits + count can tick up subtly so “spaces gather history” is felt before §03.

Global · section reveals

Each section fades/rises in on scroll (existing fadeUp), staggered children. Grain overlay + faint 48px grid persist site-wide as in-app.

Copy deck — near-final (english) Voice: personal, a little nostalgic, anti-corporate. Lowercase mono for UI/labels; sentence-case for headlines. Final wording to lock with you.
